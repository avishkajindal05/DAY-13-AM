# 🛍️ DAY-13-AM

**Product Analysis & Comparison System**

---

## 📌 Overview

This repository contains the implementation of a **Product Analysis and Comparison System** developed as part of the **Day 13 (AM Session)** assignment.

The project focuses on analyzing different categories of products—**budget, popular, and premium**—and generating insights through structured data processing and reporting.

---

## 📂 Repository Structure

```bash
DAY-13-AM/
│
├── LICENSE
├── README.md
│
├── budget_products.csv        # Budget category products
├── popular_products.csv       # Popular category products
├── premium_products.csv       # Premium category products
│
├── product_analyser.py        # Core analysis logic
├── comparison_report.py       # Generates comparison reports
│
├── Part_c.md                  # Conceptual answers / explanations
├── Part_d.ipynb               # Notebook-based analysis
├── output_report_a_b.md       # Generated report (Part A & B)
```
